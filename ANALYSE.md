## C. Recht / Легальність (НАЙВАЖЛИВІШЕ для Німеччини)

> У DE діє система **Abmahnung** — конкуренти/об'єднання можуть надсилати платні попередження за порушення. Тому легальність = пріоритет.

| Вимога | Закон | Стан | Дія власника |
|---|---|---|---|
| **Impressum** | § 5 DDG | 🟡 шаблон готовий | Заповнити: ПІБ/назва, телефон, USt-IdNr (або §19 UStG) |
| **Datenschutzerklärung** | DSGVO | 🟡 шаблон готовий | Вказати хостера; перевірити юристом |
| **Google Fonts / зовнішні ресурси** | DSGVO (LG München, Az. 3 O 17493/20) | ✅ виправлено | Шрифти локальні — **0 зовнішніх запитів**, IP не йде до Google |
| **Cookie-банер / Consent** | TDDDG | ✅ не потрібен зараз | Трекінгу немає; `localStorage` мови — технічно необхідний. **Зʼявиться реклама/аналітика → обовʼязковий CMP** |
| **Рекламні заявлення** | UWG (§5) | ✅ виправлено | Прибрано «гарантія 10 років», «100%», «сертифіковані», бренди 3M/Avery/ORACAL/KPMF |
| **Форма — згода** | DSGVO Art. 6/7 | ✅ виправлено | Додано чекбокс згоди + лінк на Datenschutz |
| **Barrierefreiheit** | BFSG (з 28.06.2025) | 🟢 ймовірно звільнені | Kleinstunternehmen-Dienstleister (<10 осіб, ≤2 млн €) звільнені; усе одно варто тримати доступність |
| **Зображення / бренди** | UrhG / MarkenG | 🟡 low-risk | AI-візуали — права ваші. На фургоні видно лого Mercedes (норм для авто-послуг, але можна замінити нейтральним) |

### Що прибрано як «spорне» (нейтралізовано):
- `10 років гарантія на PPF` → **прибрано** (Garantie = §443 BGB / UWG-ризик)
- `100% оригінальний лак` → `Захист / PPF проти сколів`
- `сертифіковані майстри/плівки` → `досвідчені майстри / якісні плівки`
- бренди `3M, Avery Dennison, ORACAL, KPMF` (стрічка + блок переваг) → нейтральні терміни послуг
- `відповідь протягом дня` → `швидка відповідь`
- «преміальні матеріали найвищого класу» → «якісні матеріали» (де було вимірюване твердження)

> ⚠️ Залишено допустимий «преміум»-тон як субʼєктивну оцінку (puffery) — це дозволено. Прибрано лише **конкретні / вимірювані** твердження без доказів.

### 🟡 Що має заповнити власник (жовті `[плейсхолдери]` у файлах):
1. `impressum.html` — ПІБ/назва, телефон, USt-IdNr
2. `datenschutz.html` — назва хостера (після вибору)
3. `agb.html` — опціонально (можна видалити, якщо не потрібні)
4. Соцмережі (Instagram/WhatsApp/Facebook) у футері та контактах
5. Перевірити Impressum+Datenschutz у юриста / через генератор (e-recht24, eRecht-Tool)

## D. SEO (для просування)

### Технічна база — ✅ готово
title/description, Open Graph + Twitter, **JSON-LD `AutoBodyShop`** (адреса, гео, послуги), hreflang ×4, sitemap.xml, robots.txt, semantic HTML, alt, lazy-load.

### 🔴 Локальне SEO — головний драйвер для München
1. **Google Business Profile** (Unternehmensprofil) — створити/підтвердити, категорія «Autofolierung / Fahrzeugfolierung», фото, години, послуги. Це №1 джерело локальних лідів.
2. **NAP-консистентність** — однакові Name/Address/Phone скрізь (сайт = GBP = каталоги).
3. **Каталоги/цитати**: Das Örtliche, Gelbe Seiten, 11880, Bing Places, Apple Maps.
4. **Відгуки** — стимулювати Google-відгуки; пізніше додати `Review`/`AggregateRating` schema (лише з реальними даними!).

### Ключові слова (DE, München-фокус)
| Кластер | Приклади |
|---|---|
| Core | `auto folieren münchen`, `fahrzeugfolierung münchen`, `car wrapping münchen` |
| PPF | `lackschutzfolie münchen`, `ppf münchen`, `steinschlagschutz folie` |
| Послуги | `vollfolierung`, `teilfolierung`, `scheibentönung münchen` (якщо роблять) |
| Бізнес | `fahrzeugbeschriftung münchen`, `werbetechnik münchen`, `firmenwagen folieren` |

### Контент (для росту органіки) — 🟡
- Окремі сторінки під послуги (Vollfolierung / PPF / Werbetechnik) з текстом 500–800 слів.
- **FAQ-блок** + `FAQPage` schema («Was kostet Folierung?», «Wie lange hält PPF?», «Ist Folierung TÜV-konform?»).
- Блог/кейси: «до/після», «скільки коштує оклейка BMW», гайди по догляду.
- Багатомовність для SEO: пізніше винести мови на окремі URL (`/de/`, `/uk/`...) замість JS-перемикача — краще індексується.

## E. Targeting / Реклама (Ads)

> Оклейка авто — **візуальний** продукт, ідеальний для Instagram/Meta та Google.

### Перед запуском будь-якої реклами — 🔴 обовʼязково:
1. **Consent Management Platform (CMP)** — банер згоди (Usercentrics, Cookiebot, Borlabs).
2. **Google Consent Mode v2** — без нього Google Ads/Analytics у ЄС працюють обмежено (з 2024).
3. **Conversion-tracking** — відправлення форми = подія конверсії (Google Ads + Meta Pixel), лише після згоди.

### Канали
| Канал | Навіщо | Нотатки |
|---|---|---|
| **Google Ads (Search)** | гарячий попит | `auto folieren münchen`, `ppf münchen`; гео-таргет München +30 км |
| **Google Ads (Performance Max)** | охоплення + Maps | потребує фото/відео-асетів |
| **Meta / Instagram Ads** | візуал, ретаргет | до/після, reels процесу; ідеально для бренду |
| **Local Services / Maps** | локальні ліди | через GBP |

### Лендинг під рекламу — статус
✅ швидкий, мобільний, є форма + чіткий CTA. 🟡 додати: клікабельний **телефон**, кнопку **WhatsApp**, та (бажано) окремі посадкові під кожну кампанію з відповідним заголовком. UTM-мітки на всі оголошення.
